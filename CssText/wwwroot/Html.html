КЛИЕНТСКАЯ ВЕБ-РАЗРАБОТКА (HTML, CSS, JAVASCRIPT)

В курсе веб-разработки мы познакоими вас с клиентской веб-разработкой, которая включает в себя верстку - обучение HTML и CSS, и написание кода на JavaScript. На курсе JavaScript будут рассмотрены синтаксис, типы данны, работа с DOM и событиями, стандартная библиотека языка.

В курсе JavaScript так же рассматривается большок количество библиотек и средств разработки - jQuery, jQuery UI, Underscore, Vue.js, Bootstrap, npm, Less/Sass, Babel. Так же обзорно рассматриваем Grunt, RequireJS, TypeScript. В качестве серверной части используем Express.js. Обучение веб-разработке закрепляется на практике.

Вы научитесь создавать клиентскую часть сайта и получите большой кругозор по клиентским технологиям.
